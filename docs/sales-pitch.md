# NIGHTOS — 夜のお店のためのワークスペース

## 「店舗が入力するだけで、キャストの売上が上がる」

---

## お店が抱える3つの課題

### 1. キャストのお客様への連絡がバラバラ
- 「いつ誰に連絡したか」が個人の記憶頼み
- ベテランが辞めると顧客情報もノウハウも消える
- 新人キャストは何をすればいいか分からない

### 2. 店舗の情報がキャストに届いていない
- 顧客の好みやNG事項を紙やLINEグループで共有 → 見落とし
- ボトルの残量を口頭で伝え忘れ → 営業チャンスのロス
- 来店情報がリアルタイムでキャストに届かない

### 3. 効果測定ができない
- どのキャストがちゃんと連絡しているか分からない
- 指名数やリピート率の推移が見えない
- 「何が売上に効いているか」が勘になっている

---

## NIGHTOSの解決策

### 店舗スタッフがやること（1日5分）

| 作業 | 所要時間 | 効果 |
|------|----------|------|
| 新規顧客を登録 | 約2分 | → キャストのカルテに即反映 |
| 来店を登録 | 約15秒 | → キャストに自動通知 |
| ボトルを登録 | 約30秒 | → 残量管理 + キャストに共有 |

**店舗スタッフは「入力するだけ」。分析も連絡の提案もAIが自動でやります。**

### キャストが得ること（入力作業なし）

| 機能 | 内容 |
|------|------|
| 🌸 今日の連絡リスト | AIが「今日連絡すべき人」を自動選定。来店間隔・誕生日・指名化チャンスを判定 |
| 📋 顧客カルテ | 店舗が入力した情報（好み・ボトル・NG事項）が自動で閲覧可能 |
| 💌 メッセージテンプレート | 顧客名・ボトル名・前回の話題が自動挿入。コピーしてLINEに貼るだけ |
| 🔮 さくらママAI | 銀座30年のママをモデルにしたAIに、接客や連絡の相談ができる |
| 📈 自分の成績 | 指名数・リピート率・目標進捗が可視化。モチベーション維持 |
| 🔔 来店通知 | 店舗が来店登録 → 30秒以内にキャストに通知 |

## 画面イメージ

### キャスト側

#### ホーム画面
- 指名数・リピート率・連絡が必要人数のサマリ
- AIが毎朝「今日の重点ポイント」を生成
- 連絡リストリスト（理由つき: 来店間隔空き / 誕生日 / 指名化チャンス）

#### 顧客カルテ
- 上部: 顧客名・職業・カテゴリ（VIP/常連/新規）
- 店舗からの共有情報（ベージュ背景・閲覧のみ）: 好きなお酒、キープボトル、店舗メモ
- 個人メモ（ピンク破線・編集OK）: 前回の話題、接客のコツ、次回の話題候補
- 来店履歴（直近5回）
- LINEスクショ取り込み → AIがメモを自動更新

#### さくらママAIチャット
- 選択式で相談開始（LINEで連絡したい / 接客中 / 営業戦略 / 自由に話す）
- 段階的にヒアリング → 顧客情報を踏まえた具体的アドバイス
- 「アドバイス + 文面例 + なぜ効くか」の3点セットで回答
- 音声入力対応

### 店舗側

#### ハブ画面
- 月間KPI（指名数・売上・連絡達成率）
- AIブリーフィング（今夜の重点ポイント）
- 顧客登録・来店登録・ボトル登録のショートカット
- キャストへの連絡機能

#### 効果ダッシュボード
- 離脱リスク顧客パネル（AIが自動検知）
- 顧客カテゴリ構成（VIP/常連/新規の比率）
- 指名数・リピート率の推移グラフ
- キャスト別成績（ARPU・連絡達成率・トップ/ボトムマーカー）

#### 管理機能
- 顧客一覧（検索・編集・削除）
- 来店履歴（確認・取消）
- ボトル管理（残量管理・1杯消費ボタン・残量警告）

## AIの力（さくらママ）

「さくらママ」は銀座の伝説的なクラブのママをモデルにしたAIです。

### できること

| 場面 | さくらママの役割 |
|------|--------------|
| 来店翌日 | 「田中さんへのお礼LINE」を顧客カルテを見ながら提案 |
| 接客中（急ぎ） | 「会話が続かない時の切り出し方」を1問で回答 |
| 営業戦略 | 「常連が離れている理由」を連絡達成率から分析 |
| ボトル残量低下 | 「次にすすめるべきボトル3本」を顧客の好みに合わせて提案 |
| LINEスクショ | スクリーンショットを貼ると、会話内容を読み取ってメモを自動更新 |
| 毎朝 | 「今日の重点ポイント」をキャストと店舗それぞれに生成 |

### さくらママの特徴
- 一般論は言わない。必ず「顧客の情報を見て」具体的に答える
- 「なぜそれが効くのか」の理由を必ず添える
- 実際に送れるLINE文面例を提案
- キャストのフィードバックを学習して、回答品質が向上

## よくある質問

**Q: スマホだけで使えますか？**
→ はい。iPhone / Android のブラウザだけで動きます。パソコンは不要です。

**Q: キャストが入力する必要はありますか？**
→ 基本的にありません。店舗が入力した情報が自動的にキャスト側に表示されます。
  キャストは「個人メモ」を書くことで、さくらママの精度が上がりますが、任意です。

**Q: 既存の予約管理システムとの連携は？**
→ 現在はスタンドアロンです。将来的にPOS連携を予定しています。

**Q: データはどこに保存されますか？**
→ クラウド（Supabase）に安全に保存されます。お店のデータは他店と共有されません。

**Q: 何店舗分のデータが見られますか？**
→ 1アカウント = 1店舗です。多店舗展開の場合は別途ご相談ください。
