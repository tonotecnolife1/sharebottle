# キャストフィードバック収集ガイド

## 目的

キャストに NIGHTOS を使ってもらい、実際の業務で役に立つかどうかの所感をヒアリングする。
「便利だった」「微妙だった」の定性情報と、利用頻度・満足度の定量情報を両方集める。

---

## ヒアリングの設計

### タイミング

| タイミング | 方法 | 目的 |
|------------|------|------|
| 使い始め（Day 1） | 対面 or LINE | 第一印象、つまづいた点 |
| 3日後 | LINE アンケート | 初期の使い勝手、機能の過不足 |
| 1週間後 | 対面 30分 | 深掘りインタビュー |
| 2週間後 | LINE アンケート + 対面 | 定着度、効果実感 |

### 聞く相手

- テスト店舗のキャスト全員（最低2〜3人）
- 経験レベルが異なる人を含める（新人 / 中堅 / ベテラン）
- 可能なら店長・ママにも（店舗側の観点）

## ヒアリング質問（コピペして使える）

### Day 1: 第一印象（5分、LINE で OK）

```
NIGHTOSの第一印象を教えてください 🌸

1. アプリを開いた時の第一印象は？
   □ 使いやすそう  □ 普通  □ 難しそう

2. ホーム画面の「今日のフォロー対象」は分かりやすかった？
   □ はい  □ まあまあ  □ よく分からなかった

3. さくらママに何か相談してみた？
   □ はい  □ まだ

4. テンプレートでLINEを送ってみた？
   □ はい  □ まだ

5. 一番気になった機能は？（自由回答）

6. 「これは使いにくい」と感じた点は？（自由回答）
```

### Day 3: 初期使い勝手アンケート（10分、LINE で送る）

```
NIGHTOS 3日間使ってみてどうでしたか？✨

【フォロー対象リスト】
1. 毎日見てる？
   □ 毎日  □ 2〜3回見た  □ 1回だけ  □ 見てない

2. リストの顧客に実際に連絡した？
   □ 複数人した  □ 1人した  □ してない

3. リストの「フォロー理由」は納得感ある？
   □ はい  □ まあまあ  □ ピンとこない

【さくらママ AI】
4. 何回使った？
   □ 5回以上  □ 2〜4回  □ 1回  □ 使ってない

5. アドバイスは参考になった？
   □ なった  □ まあまあ  □ ならなかった

6. 提案された LINE 文面はそのまま使えた？
   □ そのまま使った  □ 少し修正して使った  □ 使わなかった

7. 「こういう相談がしたかったのにできなかった」ことは？（自由回答）

【テンプレート】
8. テンプレートからLINEを送った？
   □ 何回も  □ 1〜2回  □ 使ってない

9. テンプレートの文面は自然だった？
   □ 自然  □ 少し違和感  □ AI っぽい

【総合】
10. NIGHTOS がなかった時と比べて、フォローの頻度は？
    □ 増えた  □ 変わらない  □ 減った

11. 他のキャストにもおすすめしたい？
    □ はい  □ どちらでもない  □ いいえ

12. 一番良かった機能は？（自由回答）

13. 一番いらない / 改善してほしい機能は？（自由回答）

14. その他、何でも（自由回答）
```

### 1週間後: 深掘りインタビュー（30分、対面推奨）

```
インタビューガイド（聞く側のメモ用）

■ 導入
- 最近の営業はどう？（アイスブレイク）
- NIGHTOS 使ってる？頻度は？

■ フォロー業務の変化
- NIGHTOS を使う前、フォローはどうやってた？
- 使い始めてから変わったことはある？
- 「これのおかげで助かった」エピソードはある？
- 逆に「面倒だな」と感じた瞬間は？

■ さくらママ AI
- どんな時に相談する？
- 返ってきたアドバイスで「これは良かった」と思ったものは？
- 「的外れだな」と思ったアドバイスは？
  → どこがズレてた？（トーン？内容？顧客理解？）
- さくらママの口調は好き？違和感ある？
- 選択式の相談フローはどう？テキストで打つ方がいい？

■ テンプレート
- 実際に送ったテンプレはどれ？
- 自分で修正してから送った？どこを直した？
- 「こういうテンプレが欲しい」というものはある？

■ 顧客カルテ
- カルテの情報は役に立った？
- 「この情報も載せてほしい」というものは？
- 個人メモは書いてる？書くのは面倒？

■ 音声入力
- 使った？使わなかった？
- 使った場合、認識精度はどうだった？

■ 全体
- もし明日から NIGHTOS が使えなくなったら困る？
- 有料でも使いたい？月いくらなら？
- 友達のキャストに説明するとしたら、なんて言う？
```

## フィードバックの整理方法

### スプレッドシートで管理

| キャスト名 | 日付 | カテゴリ | フィードバック内容 | 重要度 | 対応ステータス |
|-----------|------|---------|------------------|--------|-------------|
| あかり | 3/20 | さくらママ | 文面例が堅すぎる、もっと砕けた感じがいい | 高 | system prompt 修正 |
| みさき | 3/21 | テンプレ | お誕生日前日に送るテンプレが欲しい | 中 | テンプレ追加 |
| あかり | 3/22 | UI | フォロー対象の順番が分かりにくい | 中 | ソート改善 |

### カテゴリ分類

- **AI品質**: さくらママの回答精度・トーン・具体性
- **テンプレート**: 文面の自然さ・種類の過不足
- **UI/UX**: 操作性・動線・表示の分かりやすさ
- **機能要望**: 新しい機能のリクエスト
- **バグ**: 動かない・表示がおかしい
- **業務効果**: 実際の売上・指名・リピートへの影響

## フィードバックからアクションへ

### 優先度の判定基準

| 基準 | 高 | 中 | 低 |
|------|---|---|---|
| 頻度 | 複数人が言及 | 1人が言及 | — |
| 影響 | 使い続けられない | 不便 | あったら嬉しい |
| 対応工数 | 1時間以内 | 1日以内 | 数日以上 |

### 対応の流れ

```
フィードバック受領
  ↓
スプレッドシートに記録
  ↓
カテゴリ + 優先度を判定
  ↓
高優先度 → 即日〜翌日に対応
中優先度 → 週次でまとめて対応
低優先度 → バックログに積む
  ↓
対応完了 → キャストに「直しました」と報告
（= 信頼感 + 継続利用意欲の醸成）
```

## 重要な心構え

### 聞く時の注意

1. **「便利ですか？」とは聞かない** — 「はい」と言わせてしまう。代わりに「最後に使ったのはいつ？」「何を調べた？」と行動ベースで聞く
2. **沈黙を恐れない** — 考えている時間が一番重要な情報
3. **改善点を聞く時は具体的に** — 「何か不満はある？」ではなく「さくらママに相談して、返ってきた答えをそのまま使った？修正した？どこを？」
4. **「使わなかった」も重要な情報** — なぜ使わなかったのかを深掘り

### フィードバック後のフォロー

- 「教えてくれてありがとう」と必ず伝える
- 改善したら「〇〇さんの意見で直しました」と伝える
- キャストが「自分のフィードバックが反映された」と実感すると、継続的にフィードバックしてくれるようになる
