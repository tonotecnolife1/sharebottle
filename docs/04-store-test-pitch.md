# NIGHTOS テスト協力のお願い — 店舗向け説明資料

---

## NIGHTOSとは

**夜のお店のための AI ワークスペース**です。

店舗スタッフが顧客情報を入力するだけで、キャストのお客様への連絡業務をAIが支援。
指名数・リピート率を「仕組み」で上げるためのツールです。

## 具体的な業務フローの変化

### 来店時の流れ

```
【Before】
  お客様来店
    ↓
  スタッフが口頭で「田中様、A-3テーブルです」
    ↓
  キャストは誰が来たか把握しにくい
    ↓
  好みやNG事項は記憶頼み

【After（NIGHTOS）】
  お客様来店
    ↓
  スタッフがアプリで来店登録（15秒）
    ↓
  キャストのスマホに自動通知
  「田中太郎様 テーブルA-3 指名あり」
    ↓
  キャストはカルテで好み・前回の話題を確認してからテーブルへ
```

### 翌日連絡の流れ

```
【Before】
  翌朝、キャストが自分の記憶で
  「昨日来た人に連絡しなきゃ…」
    ↓
  忙しいと忘れる / 何を書くか悩む
    ↓
  連絡達成率: 30〜50%

【After（NIGHTOS）】
  翌朝、アプリのホーム画面に
  「今日の連絡リスト: 3人」
  + 各顧客の理由（来店間隔空き / 誕生日 / 指名化チャンス）
    ↓
  顧客をタップ → テンプレートを選ぶ
  → 名前・話題・ボトル名が自動挿入
  → 「コピーしてLINEへ」
    ↓
  連絡達成率: 80%+
```

### 月次の経営判断

```
【Before】
  店長が手作業で売上集計
    ↓
  どのキャストが頑張っているか「感覚」で判断
    ↓
  離脱しそうな顧客に気づかない

【After（NIGHTOS）】
  ダッシュボードを開くだけ
    ↓
  キャスト別: 指名数 / リピート率 / 連絡達成率 / ARPU が一覧
  離脱リスク顧客が自動リストアップ
  顧客カテゴリ構成（VIP/常連/新規の比率）
    ↓
  数字に基づいた判断が可能
```

## 具体的な運用方法

### 導入初日（所要時間: 約1時間）

| ステップ | 誰が | 何をする | 所要時間 |
|---------|------|---------|---------|
| 1 | 店長/スタッフ | スマホでURLを開き「ホーム画面に追加」 | 1分 |
| 2 | 店長/スタッフ | 主要顧客を10人登録（名前・好み・カテゴリ・担当） | 20分 |
| 3 | 店長/スタッフ | キープボトルを登録 | 5分 |
| 4 | 各キャスト | スマホでURLを開き「ホーム画面に追加」 | 1分/人 |
| 5 | 各キャスト | ホーム画面を確認、さくらママに1件相談してみる | 5分/人 |

### 毎日の運用（所要時間: 約10分/日）

#### 店舗スタッフ（5分/日）

| タイミング | 操作 | 所要時間 |
|-----------|------|---------|
| 新規客来店時 | 顧客登録（名前・好み・担当キャスト） | 2分 |
| 来店時 | 来店登録（テーブル選択 + 顧客選択 + 指名/フリー） | 15秒 |
| ボトルキープ時 | ボトル登録（銘柄 + オーナー + 杯数） | 30秒 |

**ポイント**: 店舗スタッフが入力する情報は**3種類だけ**。入力は最短15秒。

#### キャスト（5分/日、入力作業なし）

| タイミング | 操作 |
|-----------|------|
| 出勤前 | ホーム画面で「今日の連絡リスト」を確認 |
| 出勤前 | 連絡リストをタップ → テンプレートから LINE 送信 |
| 接客中（必要なら） | さくらママに相談（選択式 or 音声入力） |
| 閉店後 | 来店客にお礼テンプレートを送信 |
| 任意 | 個人メモを記入（次回の接客に反映） |

**ポイント**: キャストは**何も入力しなくてもアプリが使える**。カルテ情報は店舗スタッフが入力したものが自動で表示される。

## テスト協力のお願い

### テスト期間

**2週間** を想定しています。

### お店にお願いすること

1. **毎日の来店登録**（15秒/件）— スタッフに来店時の登録を習慣化してもらう
2. **キャストのアプリ利用** — 2〜3名のキャストに NIGHTOS を使ってもらう
3. **週1回のフィードバック** — 15分程度のヒアリングに応じてもらう
4. **主要顧客の初期登録** — テスト開始時に10〜20名の顧客情報を登録

### お店のメリット

- **完全無料**（テスト期間中）
- **設備投資ゼロ**（スマホだけで OK、専用端末不要）
- **フィードバックが反映される**（テスト店舗の意見で機能改善）
- **テスト終了後も継続利用可能**
- **成功事例として紹介**（許可がある場合のみ）

### お店のリスク

- 営業への影響: **なし**（既存業務に追加するだけ、置き換えではない）
- データの取り扱い: お店のデータは他店と完全に分離。テスト終了後に削除も可能
- 途中解約: **いつでも可能**。テスト期間中の途中離脱もペナルティなし

## よくある質問

**Q: スマホしかないけど使える？**
→ はい。iPhone / Android のブラウザだけで動きます。アプリのインストールは不要です。

**Q: キャストが嫌がりませんか？**
→ キャストの入力作業はゼロです。「見るだけ」で使えます。LINE のテンプレートは「コピペで送れる」ので、むしろ楽になります。

**Q: お客様の個人情報は大丈夫？**
→ 名前・誕生日・好みの程度。住所や電話番号は扱いません。データはクラウドに暗号化保存され、他店からはアクセスできません。

**Q: テスト中に費用はかかる？**
→ 完全無料です。AI 機能の API 費用（月額1,000円程度）もこちらで負担します。

**Q: 既存のPOSシステムと競合する？**
→ しません。NIGHTOS は「連絡支援 + AI アドバイス」に特化しており、会計やPOSの機能はありません。並行利用が前提です。

**Q: テスト後に「やっぱりいらない」と言える？**
→ はい。テスト期間後の継続は任意です。データの削除もリクエストに応じます。
