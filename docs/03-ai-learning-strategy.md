# さくらママ AI 学習・品質改善戦略

## 目的

元 No.1 キャバ嬢やママの実体験・ノウハウをさくらママ AI に反映し、
「一般的な AI アドバイス」ではなく「夜の世界を知っている人の具体的なアドバイス」を返せるようにする。

---

## さくらママの品質を決める3つの要素

```
┌─────────────────────────────────────┐
│  1. System Prompt（人格と知識）      │ ← 大枠のルール
│  2. Few-shot Examples（回答例）      │ ← 具体的なお手本
│  3. Context Prefix（顧客情報）       │ ← 個別の状況把握
└─────────────────────────────────────┘
```

### 1. System Prompt を改善する（即効性: 高）

今の system prompt には「夜の世界の常識」が箇条書きで入っている。
ここに**プロの暗黙知**を追加するのが最も効果的。

**例**: 今あるルール
```
- 来店間隔が通常の1.5倍を超えたら離脱リスク
```

**プロの知見を追加した後**:
```
- 来店間隔が通常の1.5倍を超えたら離脱リスク
- ただしVIPは「忙しいだけ」の場合が多い。焦って連絡するより
  「覚えてますよ」を伝える軽い一言が効く
- 新規客の来店間隔が空いた場合は「他の店に行った」可能性が高い。
  その場合は話題を変えてアプローチ（前回と同じ切り口は逆効果）
- 季節の変わり目（GW前、お盆前、年末）は自然に間隔が空く時期。
  この時期の離脱は過度に心配しなくてよい
```

### 2. Few-shot Examples を増やす（即効性: 高）

今は2つの例（フォロー相談 + 接客中相談）がある。
プロに「こういう相談にはこう答えるべき」というお手本を 5〜10 個もらって追加する。

**追加すべきカテゴリ**:
- 指名化の具体戦略（新規3回目、5回目、10回目の壁）
- VIP離脱防止の対応
- 他の子に指名が移りそうな時の対処
- 接待客 vs 個人客の対応差
- お酒を飲めないキャストの立ち回り
- 同伴・アフターの誘い方
- 誕生日・記念日の演出
- クレーム対応（機嫌が悪い客）

### 3. Context の充実（効果: 中）

現在は顧客カルテの情報を渡している。追加できる文脈:
- 「この顧客の過去3回のさくらママ回答 + フィードバック」
  → キャストが「参考になった」と言った回答の傾向を反映
- 「この店舗の平均客単価・客層」
  → 銀座 vs 六本木 vs 赤坂でアドバイスのトーンを変える
- 「現在の季節・曜日・時間帯」
  → 金曜夜の接客相談 vs 月曜朝のフォロー相談で回答が変わるべき

## プロへのヒアリング方法

### 誰に聞くか

| 対象 | 得られる知見 | 優先度 |
|------|------------|--------|
| 元 No.1 キャバ嬢 | 指名を取る具体戦略、LINE テクニック、VIP 対応 | 最高 |
| 現役ママ | 店舗経営視点、キャスト教育、顧客管理の勘所 | 最高 |
| 現役キャスト（経験3年以上） | 日常の接客判断、失敗談、成功体験 | 高 |
| 店舗オーナー | 数字（売上・指名・リピート率）で見る成功パターン | 中 |

### ヒアリングの進め方

#### セッション1: シチュエーション別回答の収集（60分）

**目的**: さくらママが返すべき「お手本回答」を集める

**進め方**:
1. NIGHTOS のさくらママ画面を見せる
2. 「こういう相談が来たら、何て答えます？」と聞く
3. プロの回答を録音 or メモ
4. 10〜15 シチュエーションを網羅

**シチュエーション例（コピペして使う）**:

```
1. 新規のお客様が3回来てくれました。まだフリーです。
   次回、指名してもらうにはどうしたらいいですか？

2. VIPのお客様が2週間来てません。普段は毎週来てくれます。
   LINEでどうフォローしたらいいですか？

3. 常連のお客様の誕生日が来週です。
   何か特別なことをしたいんですが、どうしたらいいですか？

4. 接客中、お客様の機嫌が急に悪くなりました。
   今どうしたらいいですか？

5. 他の子に指名が移りそうな気配があります。
   どうやって取り戻しますか？

6. お客様が「もう来ないかも」と言いました。
   どう対応しますか？

7. 新規のお客様にボトルを入れてもらいたいです。
   いつ、どうやって提案しますか？

8. 同伴に誘いたいお客様がいます。
   LINE でどう切り出しますか？

9. お客様との LINE で話題が尽きました。
   何を送ればいいですか？

10. お客様が仕事の愚痴ばかりで疲れます。
    どう対応すれば指名は維持できますか？
```

#### セッション2: 既存回答のレビュー（30分）

**目的**: さくらママの今の回答のどこがダメかを指摘してもらう

**進め方**:
1. さくらママに実際に質問を投げる（プロの目の前で）
2. 返ってきた回答を見せて「これ、現場で通用しますか？」と聞く
3. 「ここはいい」「ここはダメ」「こう直すべき」を記録

**チェックポイント**:
- トーンは自然か？（堅すぎ？軽すぎ？）
- 具体性は十分か？（一般論になってないか？）
- 文面例はそのまま送れるレベルか？
- 「なぜ効くか」の説明は納得感があるか？
- 夜の世界の常識に反してないか？

#### セッション3: 暗黙知の抽出（60分）

**目的**: プロが無意識にやっていることを言語化する

**進め方**:
```
- 「一番売上が伸びた時期はいつですか？何を変えましたか？」
- 「新人キャストに最初に教えることは何ですか？」
- 「お客様の種類を分類するとしたら、どう分けますか？」
- 「LINE で絶対にやってはいけないことは？」
- 「指名が取れるキャストと取れないキャストの一番の違いは？」
- 「お酒の席で一番大事なことは何ですか？」
- 「常連を離さないために、日常的にやっていることは？」
- 「VIPのお客様と一般のお客様で、対応をどう変えますか？」
```

## 収集した知見の反映方法

### Step 1: 知見を構造化する

ヒアリング結果を以下のフォーマットに整理:

```yaml
category: "指名化戦略"
situation: "新規3回目のフリー客を指名化したい"
insight: |
  直接「指名してね」は絶対NG。代わりに「次も私に会いに来て」
  というニュアンスで「続きを話す約束」を作る。帰り際に
  「次はこれを試してほしい」と具体的なフックを残す。
example_line: |
  「○○さん、今日のお話すごく楽しかった。
  次いらした時に、この前話してた○○の続き聞かせてくださいね。
  私も調べておきますから😊」
why_it_works: |
  人は「約束を果たさないといけない」という心理が働く。
  指名というハードルではなく「続きを話す」という軽い約束に
  すり替えることで、自然に次回も同じキャストを呼ぶ。
source: "元No.1 Aさん"
```

### Step 2: System Prompt に反映

構造化した知見を system prompt の「夜の世界の常識」セクションに追加:

```
ファイル: features/ruri-mama/data/system-prompt.ts
```

1つの知見 = 2〜3行で追加。prompt 全体が長くなりすぎないよう、最も汎用的なものを選ぶ。

### Step 3: Few-shot Examples に反映

特に良い回答例は few-shot example として追加:

```
現在: 2つの例（フォロー + 接客中）
目標: 5〜8つの例（主要シチュエーションをカバー）
```

追加候補:
1. VIP 離脱防止
2. 指名化（新規 → 指名）
3. 誕生日/記念日対応
4. ボトル提案
5. クレーム/機嫌対応

### Step 4: フィードバックループ

```
プロの知見 → System Prompt 更新 → キャスト利用
    ↑                                    ↓
    ← ←  キャストのフィードバック ← ← ←
```

キャストの「参考になった/ならなかった」データが蓄積されると:
- 「参考になった」が多い回答パターンを分析 → 強化
- 「ならなかった」が多い回答パターンを分析 → 修正
- 定期的に system prompt を改訂（月1回程度）

## 品質評価の指標

### 定量指標

| 指標 | 測定方法 | 目標 |
|------|----------|------|
| 参考になった率 | 「参考になった」÷ 全評価数 | 70%以上 |
| 文面そのまま使用率 | ヒアリングで聞く | 50%以上 |
| さくらママ利用頻度 | API コールログ | 1キャスト1日3回以上 |
| テンプレ経由のLINE送信率 | follow_logs | 1キャスト1日2件以上 |

### 定性指標

- 「さくらママの言う通りにしたら指名が取れた」エピソード
- 「テンプレのおかげで連絡が楽になった」声
- 「これなしでは営業できない」レベルの依存度
